Face Recognition Attendance System

An AI-based attendance management system that automatically detects and recognizes faces using computer vision and marks attendance.

Project Overview

This project is an automated attendance management system built using Python, OpenCV, and Scikit-Learn. The system captures faces through a webcam, recognizes registered users using a KNN machine learning model, and automatically records attendance in CSV files. It removes the need for manual attendance and improves efficiency.

Features

  • Real-time face detection using OpenCV
  • Face recognition using KNN machine learning algorithm
  • Register new users by capturing face data
  • Automatic attendance recording
  • Attendance stored in CSV files
  • Simple dashboard using Streamlit

How the System Works

Step 1: Register Face

User registers by capturing face images through the webcam. The system stores the face dataset for future recognition.

Step 2: Face Detection

The system uses OpenCV Haar Cascade to detect faces from the webcam in real time.

Step 3: Face Recognition

The system compares detected faces with stored data using the KNN machine learning algorithm.

Step 4: Attendance Marked

Once the face is recognized, the system automatically records attendance with name and timestamp in a CSV file.

Tech Stack

  • Python
  • OpenCV
  • NumPy
  • Scikit-Learn (KNN)
  • Pandas
  • Streamlit
