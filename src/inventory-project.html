Smart Retail Inventory & Billing System

A Spring Boot based inventory management and billing system that allows users to add products, manage stock, and generate bills.

Project Overview

This project is a full-stack inventory and billing management system built using Spring Boot and MySQL. It allows users to add products, manage stock, and generate bills while automatically updating inventory quantities.

Features

  • Add new Products
  • View and manage product inventory
  • Create bills using product quantity
  • Automatic stock update after billing
  • Bill history stored in database

How the System Works

Step 1: View Product Inventory

User can view all available products with category, price, and quantity.

Step 2: Add New Product

Admin can add a new product by entering product name, category, price, and quantity.

Step 3: Create Bill

User selects products and enters quantity to generate a bill. The system checks available stock automatically.

Step 4: Bill Generated

After generating the bill, the system automatically updates the product stock and stores bill details in the database.

Tech Stack

  • Java
  • Spring Boot
  • Thymeleaf
  • MySQL
  • HTML & CSS
